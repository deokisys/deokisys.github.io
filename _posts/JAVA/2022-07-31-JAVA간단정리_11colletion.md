# collection
- list계열
  - list, set
- map계열
  - map 

- add
- contains, equals, isEmpty, size
- clear
- toArray


## list
- list(arraylist,linkedlist, stack), queue
- 순서가 존재
- 중복 허락
- get, indexof, remove(index), set(index, e)수정
- 조회 빠름
- 새로운 배열을 만드는 형식
- 비순차 추가, 삭제 오래걸림
  - 순차는 빠름
- 삭제시 list크기 바뀌는거 조심

### linked list
- 순차 추가삭제 느림
  - 비순차는 빠름
- 조회 느림

## set
- 순서 없음
- 중복 없음
- 수정 없음
  - 지우고 추가하면 되니까
- 동일 기준
  - equeals, hashcode가 동일


## map
- 순서 없음
- 키 중복 없음
- key, value
- put
  - putIfAbsent - 없으면 추가하기
- containsKey, containsValue, keySet,get(키),vlaues,size,isEmpty
- clear, remove(키)
- put- 수정도 가능
